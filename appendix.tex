Python package (\texttt{src/dispersion}) with a top-level pipeline script
(\texttt{main.py}) and a pytest suite (\texttt{tests/}). Code is grouped
below by pipeline stage: data acquisition, signal construction, RMT
cleaning, the backtest engine, the ML layer, shared utilities, then tests.
